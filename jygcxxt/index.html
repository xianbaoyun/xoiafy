基因工程技术学习通答案

 •   阅读量  -  •  阅读时间: 7.2分
Fang18.  2021-11-09


请核对，是这门课程！

1.1.1

1.【单选题】组成DNA的骨架成分是( )
A、碱基与戊糖
B、碱基与磷酸
C、碱基与碱基
D、戊糖与磷酸
我的答案：A

2.【单选题】有关RNA的叙述,错误的是( )
A、主要有mRNA、tRNA和rRNA三类
B、胞质中只有mRNA、tRNA
C、tRNA是细胞内分子量最小的一种RNA
D、rRNA可与蛋白质结合
我的答案：B

3.【单选题】DNA 碱基组成的规律是( )
A、[A]=[C]; [T]=[G]
B、[A]+[T]=[C]+[G]
C、[A]=[T]; [C]=[G]
D、([A]+[T])/([C]+[G])=1
我的答案：C

4.【单选题】1953年Watson和Crick提出( )
A、DNA链通过氢键连接成一个双螺旋
B、三个连续的核苷酸代表一个密码子
C、遗传物质通常是DNA而非RNA
D、分离到回归突变体证明这一突变并非一个缺失突变
我的答案：A

5.【单选题】DNA的二级结构指的是( )
A、4种核苷酸的连接及其排列顺序,表示了DNA分子的化学组成
B、DNA双螺旋进一步扭曲盘旋所形成的负螺旋结构
C、两个DNA分子反向平行盘绕所生成的双螺旋结构
D、染色质结构
我的答案：C

6.【单选题】生物体内天然状态的DNA几乎都以( )形式存在。
A、B-DNA
B、Z-DNA
C、A-DNA
D、cDNA
我的答案：A

7.【单选题】下列几种DNA分子中的碱基组成比例各不相同,哪一种DNA的解链温度(Tm)最低( )
A、DNA中A+T含量占15%
B、DNA中G+C含量占25%
C、DNA中G+C含量占40%
D、DNA中A+T含量占60%
E、DNA中G+C含量占70%
我的答案：B

8.【单选题】下面不属于RNA与DNA结构上的区别的是( )。
A、核糖非脱氧核糖
B、尿嘧啶(U)非胸腺嘧啶(T)
C、通常以单链分子存在,可自身折叠形成局部发夹或茎环结构
D、通常形成双螺旋以双链形式存在
我的答案：D

9.【多选题】下面哪些属于RNA与DNA结构上的区别( )
A、核糖非脱氧核糖
B、尿嘧啶(U)非胸腺嘧啶(T)
C、通常以单链分子存在,可自身折叠形成局部发夹或茎环结构。
D、除了“A=U”和“G≡C”配对之外,还存在一些非典型的配对,如“G=U”配对等。
我的答案：ABCD

10.【多选题】下面哪些属于遗传物质。( )
A、DNA
B、RNA
C、糖类
D、脂类
我的答案：AB

1.【单选题】CTAB在植物组织DNA提取过程中的主要作用是( )
A、CTAB可以溶解细胞膜,并结合核酸,使核酸便于分离。
B、CTAB可以使DNA沉淀
C、CTAB可以使DNA溶解
D、CTAB可以使蛋白溶解,并结合核酸,使核酸便于分离。
我的答案：A

2.【单选题】CTAB的中文全称是( )
A、十六烷基三甲基溴化铵

B、75%乙醇可以沉淀DNA
C、75%乙醇可以溶解DNA
D、75%乙醇可以沉淀上清液中的杂质
我的答案：A

A、异丙醇可以沉淀DNA
B、异丙醇可以溶解DNA
C、异丙醇可以溶解上清液中的杂质
D、异丙醇可以沉淀上清液中的杂质

2.1.1

1.【单选题】关于碱解法分离质粒DNA,下面哪一种说法不正确( )
A、溶液I的作用是悬浮菌体
B、溶液II的作用是使DNA变性
C、溶液III的作用是使DNA复性
D、质粒DNA分子小,所以没有变性,染色体变性后不再复性
我的答案：D

2.【单选题】在基因工程中,通常使用的质粒存在于( )
A、细菌染色体
B、酵母染色体
C、细菌染色体外
D、以上都不是
我的答案：C

3.【单选题】常用的小量制备质粒DNA的方法不包括:( )
A、煮沸法
B、碱裂解法
C、去污剂法
D、层析法
我的答案：D

4.【单选题】碱裂解法提取质粒DNA时,氯仿/酚的作用是:( )
A、去除染色体DNA
B、去除脂类物质
C、去除杂蛋白
D、去除RNA
我的答案：C

5.【单选题】碱裂解法分离提取质粒DNA的基本原理是:( )
A、碱性条件下染色体DNA变性,去除变性条件后,可以复性;而质粒DNA则相反。
B、碱性条件下质粒DNA变性,去除变性条件后,可以复性;而染色体DNA则相反。
我的答案：A

6.【单选题】制备的质粒DNA在琼脂糖凝胶电泳时,通常以三条带形式出现,不包括下列哪种:( )
A、线性
B、开环
C、闭环超螺旋
D、单链DNA
我的答案：D

1.2.2

1.【单选题】质粒DNA提取分离中,solution I的作用是:( )
A、裂解菌体,使DNA分子变性
B、悬浮菌体,稳定菌体性质
C、中和碱性,使DNA分子复性
D、去除杂蛋白
我的答案：B

2.【单选题】提取分离质粒DNA时,加入的2倍体积的冰预冷的无水乙醇的目的是:( )
A、去除杂蛋白
B、去除脂类物质
C、沉淀质粒DNA
D、沉淀RNA
我的答案：C

3.【单选题】质粒DNA提取分离中,solution II的作用是:( )
A、裂解菌体,使DNA分子变性
B、悬浮菌体,稳定菌体性质
C、中和碱性,使DNA分子复性
D、去除杂蛋白
我的答案：A

4.【单选题】在质粒提取实验中,solution III处理后,离心进行分离,此时,质粒DNA存在于:( )
A、上清液
B、沉淀
我的答案：A

5.【单选题】质粒DN
A在琼脂糖凝胶电泳时,电泳迁移率按照从大到小的顺序排列,正确的是:( )
A、线性DNA>闭环DNA>超螺旋DNA
B、闭环DNA>线性DNA>超螺旋DNA
C、超螺旋DNA>闭环DNA>线性DNA
D、超螺旋DNA>线性DNA>闭环DNA
我的答案：B
